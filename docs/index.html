Quick Setup

1

Clone the Repository

git clone https://github.com/dollspace.gay/Aurora-Prism.git
cd PublicAppView
2

Configure Environment

cp .env.example .env
# Edit .env with your settings:
# - DATABASE_URL
# - APPVIEW_DID (your did:web: identifier)
# - ENCRYPTION_KEY (generate with: openssl rand -base64 32)
3

Generate OAuth Keys

bash oauth-keyset-json.sh
			
 bash setup-did-and-keys.sh
# This creates oauth-keyset.json, did document and private keys
4

Start with Docker

docker-compose up -d
# Your AppView will be available at http://localhost:5000
